4 Forms

Bootstrap Buttons and Forms with some customisations.

4.1 Buttons View in a new window

Package: @dpe/buttons

See Bootstrap buttons for usage. Some styles have been overridden as demonstrated below.

4.1.1 Button groups View in a new window

See Bootstrap button groups for documentation.

Twig: templates/button-group.twig

Example
Markup 
<div class="btn-group" role="group" aria-label="Basic example">
  <button type="button" class="btn btn-primary">Left</button>
  <button type="button" class="btn btn-primary">Middle</button>
  <button type="button" class="btn btn-primary">Right</button>
</div>

4.1.2 Icon integration View in a new window

Extend the Bootstrap buttons with our icon library. Simply add the .btn-icon-[left|right] class along with any icon class from the available set.

Make sure you have the @dpe/icons package installed and are including it's styles.

Twig: templates/icon-button.twig

Default example
Small button Button Large button
The left sided icon: btn-icon-left
Small button Button Large button
The right sided icon: btn-icon-right
Small button Button Large button
Markup 
<a class="btn btn-primary btn-sm {{ modifier_class }} icon--download-white" href="javascript:void(0)" role="button">Small button</a>
<a class="btn btn-primary {{ modifier_class }} icon--download-white" href="javascript:void(0)" role="button">Button</a>
<a class="btn btn-primary btn-lg {{ modifier_class }} icon--download-white" href="javascript:void(0)" role="button">Large button</a>

4.2 Forms View in a new window

Package: @dpe/form

See Bootstrap forms for usage details. Some styles have been overridden as demonstrated below.

4.2.1 Checkbox and Radios View in a new window

Extend the Bootstrap radios and checkbox styling with the .form-check-fancy class.

Twig: templates/checkbox-radio.twig

Example
Markup 
<fieldset class="form-group">
  <legend></legend>
  <div class="form-check form-check-fancy">
    <input class="form-check-input" type="checkbox" value="" id="defaultCheck1">
    <label class="form-check-label" for="defaultCheck1">
      Default checkbox
    </label>
  </div>
  <div class="form-check form-check-fancy">
    <input class="form-check-input" type="checkbox" value="" id="defaultCheck2" checked>
    <label class="form-check-label" for="defaultCheck2">
      Checked checkbox
    </label>
  </div>
  <div class="form-check form-check-fancy">
    <input class="form-check-input" type="checkbox" value="" id="defaultCheck3" disabled>
    <label class="form-check-label" for="defaultCheck3">
      Disabled checkbox
    </label>
  </div>
</fieldset>
<fieldset class="form-group">
  <div class="form-check form-check-fancy">
    <input class="form-check-input" type="radio" name="exampleRadios" id="exampleRadios1" value="option1">
    <label class="form-check-label" for="exampleRadios1">
      Default radio
    </label>
  </div>
  <div class="form-check form-check-fancy">
    <input class="form-check-input" type="radio" name="exampleRadios" id="exampleRadios2" value="option2" checked>
    <label class="form-check-label" for="exampleRadios2">
      Checked radio
    </label>
  </div>
  <div class="form-check form-check-fancy">
    <input class="form-check-input" type="radio" name="exampleRadios" id="exampleRadios3" value="option3" disabled>
    <label class="form-check-label" for="exampleRadios3">
      Disabled radio
    </label>
  </div>
</fieldset>

4.2.5 Stacked form View in a new window

Example
We'll never share your email with anyone else.
Markup 
<form>
  <div class="form-group">
    <label for="exampleInputEmail1">Email address</label>
    <input type="email" class="form-control" id="exampleInputEmail1" aria-describedby="emailHelp" placeholder="Enter email">
    <div id="emailHelp" class="form-text text-muted">We'll never share your email with anyone else.</div>
  </div>
  <div class="form-group">
    <label for="exampleInputPassword1">Password</label>
    <input type="password" class="form-control" id="exampleInputPassword1" placeholder="Password">
  </div>
  <div class="form-group form-check form-check-fancy">
    <input type="checkbox" class="form-check-input" id="exampleCheck1">
    <label class="form-check-label" for="exampleCheck1">Check me out</label>
  </div>
  <button type="submit" class="btn btn-primary">Submit</button>
</form>

4.2.6 Icon integration View in a new window

Extend the Bootstrap form controls with our icon library. Simply add the .form-control-icon class along with any icon class from the available set.

Make sure you have the @dpe/icons package installed and are including it's styles.

Twig: templates/icon-controls.twig

Example
Markup 
<div class="form-group">
  <label for="exampleFormControlIconInput1">Email address</label>
  <input type="email" class="form-control form-control-icon icon--email" id="exampleFormControlIconInput1" placeholder="name@example.com">
</div>
