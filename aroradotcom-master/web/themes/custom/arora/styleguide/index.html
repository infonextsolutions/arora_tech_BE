Component Library

Status

Current status: Pre-alpha 1.0-dev release

This document, as it stands now, allows us to preview and test design components that are under active design and development.

Framework

Bootstrap version: 4.1.3

This component library consumes the Bootstrap framework via NPM. It overrides certain aspects of Bootstrap and adds some custom components.

Instructions

The recommended way to consume this Component Library is to include the relevant packages via NPM, then import the Sass or compiled CSS into your own styles.

See the STARTERKIT project for detailed instructions on this approach.

Installing via NPM

All packages are scoped with the @dpe namespace.

To install the latest release of a package run:

$ npm install @dpe/base --save-dev

To install a specific release of a package run:

$ npm install @dpe/base@1.0.2 --save-dev

The base package is the minimum requirement, which requires the @dpe/builder package. All other packages require the @dpe/base package (and therefore also @dpe/builder) as they all import the @dpe/base/src/global file. In addition to this many components depend on styles and utility classes defined in @dpe/base/src/base to display properly.

Several components also require the @dpe/layout and @dpe/icon classes to be present.

Packages

Package Description Example
@dpe/builder Build files to allow other packages to compile outside of the monorepository
@dpe/base Atomic level styles and global variables Base
@dpe/layout Flexbox based Grid system and Layout utility classes Layout
@dpe/icons Custom icon library classes Icons
@dpe/buttons Button styles with Icon integration Buttons
@dpe/form Base form styles with custom Radio/Checkbox styles Forms
@dpe/header Header, search and mobile navigation styles Header
@dpe/footer Footer and Prefooter styles Footer
@dpe/navigation All navigation - Main menu, Breadcrumbs, Pagination, Social links etc Navigation
@dpe/accordion An accordion (expand/collapse) component Accordion
@dpe/banner Hero banner and Carousel components with variations Banner + Carousel
@dpe/teasers A card component with variations and Section title Teasers + Section title
@dpe/image-grid An image grid with gallery/lightbox component Image grid
@dpe/tabbed A vertical tabs component Tabbed
