3 Icons

Package: @dpe/icons

A custom icon library for use as utility classes or with the icon() mixin.

Use the utility class by prefixing any icon name with icon--. For example .icon--alert. These classes integrate with the Buttons and Form packages, or as a standalone element with the addition of .icon
