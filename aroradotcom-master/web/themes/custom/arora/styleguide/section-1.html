1.2 Fonts View in a new window

The Gotham webfonts are loaded from https://cloud.typography.com/6161932/7684812/css/fonts.css and are available for use on certain whitelisted domains. System fonts (like this) are used as the fallback.

The regular (400) weight uses Gotham Book and is available in normal and italic. The bold (700) weight uses Gotham Bold and is only available in normal.

Font variables:

Name Value
$font-family-base 'Gotham A', 'Gotham B', $font-family-sans-serif
$font-weight-normal 400
$font-weight-bold 700

1.3.2 icon() View in a new window

A mixin to include an icon by name. You can also override the background-size property. eg. @include icon('alert', 20px); See the Icons package for the full name set.

Parameters:
  • $name
    Icon name (filename without the .svg extension)
  • $size
    Set a custom background size property
    Defaults to: FALSE

1.3.3 image-replace() View in a new window

Modern image replacement technique, available as a mixin only rather than as a utility class. Also available as image-replace-reset() to undo image-replace()

Included since bootstrap is deprecating .hide-text

Parameters:
  • $display
    Sometimes what we're replacing can be an inline-block element.
    Defaults to: block

Responsive sizing

Responsive sizing is done on a per element basis. The standard pattern is to increase by one size on desktop for all body and heading sizes. The example column refers to the desktop size.

Typographic utility classes

All heading and body sizes have a corresponding class that can be used on any element. The class name structure follows the same pattern as the names here, only with a 2 dashes instead of 1. So .heading--xxl or .body--xs for example. These class have the responsive sizing baked into them.
