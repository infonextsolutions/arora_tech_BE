7.4 Image grid View in a new window

Package: @dpe/image-grid

Combines the Carousel and Bootstrap Modal components to create a lightbox style image gallery.

Bootstrap javascript:

Requires:

  • /js/carousel.js
  • /js/modal.js

Dependencies:

  • /js/jquery.min.js
  • /js/util.js

Usage: See Bootstrap documentation for both the Modal and Carousel

Twig: image-grid.twig

Example
Previous Next
Markup 
{% set images = images|default(['396', '513', '549', '821', '579', '839', '611', '536', '672', '667', '820', '838']) %}
<div class="page-container">
  <ul class="list-unstyled row">
    {% for img in images %}
      <li class="col-sm-2 mb-4">
        <a href="#imageGridExample" data-toggle="modal" data-slide-to="{{ loop.index0 }}">
          <img class="img-fluid rounded" src="https://picsum.photos/170?image={{ img }}" alt="Image grid thumbnail #{{ loop.index }}" />
        </a>
      </li>
    {% endfor %}
  </ul>

  <div class="modal fade carousel slide" id="imageGridExample" tabindex="-1" role="dialog" aria-label="Image Gallery" aria-hidden="true">
    <div class="modal-dialog modal-dialog-centered modal-lg">
      <div class="modal-content">
        <div class="modal-body">
          <ol class="carousel-indicators">
            {% for img in images %}
              <li data-target="#imageGridExample" data-slide-to="{{ loop.index0 }}" class="{{ loop.first ? 'active' }}"></li>
            {% endfor %}
          </ol>
          <div class="carousel-inner">
            {% for img in images %}
              <div class="carousel-item {{ loop.first ? 'active' }}">
                <img class="img-fluid rounded" src="https://picsum.photos/1182/762?image={{ img }}" alt="Image grid placeholder #{{ loop.index }}">
              </div>
            {% endfor %}
          </div>
          <a class="carousel-control-prev" href="#imageGridExample" role="button" data-slide="prev">
            <span class="carousel-control-prev-icon" aria-hidden="true"></span>
            <span class="sr-only">Previous</span>
          </a>
          <a class="carousel-control-next" href="#imageGridExample" role="button" data-slide="next">
            <span class="carousel-control-next-icon" aria-hidden="true"></span>
            <span class="sr-only">Next</span>
          </a>
          <button type="button" class="btn-close" data-dismiss="modal">
            <span class="icon icon--close-black"></span>
            <span class="sr-only">Close</span>
          </button>
        </div>
      </div>
    </div>
  </div>
</div>
